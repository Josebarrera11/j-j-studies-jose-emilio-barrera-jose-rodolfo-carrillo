Temas del Bimestre 2

Hardware

El hardware es el conjunto de componentes físicos de una computadora o dispositivo electrónico, como la CPU, la memoria, el disco duro y la placa base. Es responsable de ejecutar las instrucciones del software y realizar tareas esenciales, desde el procesamiento de datos hasta el almacenamiento y la conectividad.

Software

El software es el conjunto de programas y aplicaciones que permiten a un dispositivo ejecutar tareas y procesar datos. Incluye sistemas operativos, aplicaciones y herramientas que gestionan el hardware y permiten la interacción del usuario. El software es esencial para la funcionalidad de cualquier dispositivo digital.
